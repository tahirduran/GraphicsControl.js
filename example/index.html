Creating a Bezier

We need four ControlPoints to add Bezier.

<canvas id="CanvasBezier"></canvas>
<script>
  var g = new GraphicsControl ("CanvasBezier");
  var p1 = g.AddControlPoint(80,150);
  var p2 = g.AddControlPoint(120,40);
  var p3 = g.AddControlPoint(230,160);
  var p4 = g.AddControlPoint(270,50);
  g.AddBezier(p1,p2,p3,p4);
</script>

Fill Rectangle

The Points control the head and the end of the rectangle. We need to BeginFill to fill in drawing.

<canvas id="CanvasRect"></canvas>
<script>
  var g = new GraphicsControl ("CanvasRect");
  var p1 = g.AddControlPoint(80,50);
  var p2 = g.AddControlPoint(270,150);
  g.BeginFill("skyblue");
  g.AddRect(p1,p2);
  g.EndFill();
</script>

Polygon

We need to use a ControlPoints Array to add the Polygon.

<canvas id="CanvasPolygon"></canvas>
<script>
  var g = new GraphicsControl ("CanvasPolygon");
  var p = [ g.AddControlPoint(60,150),
            g.AddControlPoint(120,40),
            g.AddControlPoint(230,40),
            g.AddControlPoint(290,150) ];
  g.BeginFill("lightgreen");
  g.AddPolygon(p);
  g.EndFill();
</script>

Input Connect

We can connect ControlPoints to Inputs using ConnectInput.

<canvas id="CanvasInput"></canvas>
<p class="input-title">
  x:<input id="Input1" type="number" value="0">
  y:<input id="Input2" type="number" value="0">
  x:<input id="Input3" type="number" value="0">
  y:<input id="Input4" type="number" value="0">
</p>
<script>
  var g = new GraphicsControl ("CanvasInput");
  var p1 = g.AddControlPoint(80,150);
  var p2 = g.AddControlPoint(270,50);
  p1.ConnectInputX("Input1");
  p1.ConnectInputY("Input2");
  p2.ConnectInputX("Input3");
  p2.ConnectInputY("Input4");
  g.AddLine(p1,p2);
</script>

Control Limit

We can limit the ControlPoints as in the following example.

<canvas id="CanvasLineLimit"></canvas>
<script>
  var g = new GraphicsControl ("CanvasLineLimit");
  g.AddRect(125,50,225,150);
  var p1 = g.AddControlPoint(125,150);
  var p2 = g.AddControlPoint(225,50);
  p1.PositionLimit(125,50,225,150);
  p2.PositionLimit(125,50,225,150);
  g.AddLine(p1,p2);
</script>

Direction Limit

We can limit the ControlPoints in one direction. DirectPositionLimit is used for DirectionLimit.

<canvas id="CanvasDirectionLimit"></canvas>
<script>
  var g = new GraphicsControl ("CanvasDirectionLimit");
  var p1 = g.AddControlPoint(125,150);
  var p2 = g.AddControlPoint(225,50);
  p1.DirectionLimit(1,-1);
  p2.DirectionLimit(1,-1);
  p1.DirectPositionLimit(-50,50);
  p2.DirectPositionLimit(-50,50);
  g.AddLine(p1,p2);
</script>
